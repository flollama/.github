# Flollama
## Overview
Building open and accessible AI systems.

Flollama is an open AI initiative founded by [Pratyush Kumar](https://github.com/pratyush0898).

The organization focuses on developing full-stack AI applications, language model integrations, and developer tools that are accessible, transparent, and practical to use.

Flollama is not limited to a single product. It is an evolving collection of projects exploring modern AI systems, interfaces, and infrastructure.

---

## Core Project

### Flollama Chatbot

A full-stack conversational AI application built with modern web technologies.

- Powered by Gemini 2.0 Flash Lite (previously LLaMA-based models)
- Built using Next.js, React, and TailwindCSS
- Integrated with Firebase Authentication and Firestore
- Designed for performance, usability, and clean interface design

Main repository:  
https://github.com/flollama/flollama

Live application:  
https://flollama.vercel.app/
